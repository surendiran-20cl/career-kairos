# NOTE: spacy, scikit-learn, and numpy versions here are newer than
# this project's original pins. This was required because Python
# 3.12.4 breaks spacy 3.7.4's internals (a known upstream bug -

# fixed that, which in turn required numpy 2.x, which in turn
# required upgrading scikit-learn past 1.3.2 (which hard-pins
# numpy<2.0). All three were tested together and confirmed working
# end-to-end (resume upload + parsing + matching) after this change.
